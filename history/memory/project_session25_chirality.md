**Session 25 (2026-03-28): unique homomorphism, traceless commutator, chirality**

Derived results (added to main body):
- tr: u(d) → u(1) is the unique Lie algebra homomorphism to a 1-dimensional target. Any homomorphism to abelian target kills [u(d),u(d)] = su(d). The trace is the only scalar-valued information a write carries.
- tr([A,B]) = 0: commutator is traceless. Ordering (non-abelian, visible to L) and conservation (u(1), invisible to L) are algebraically orthogonal. Ordering enters conserved phase only through state-dependence, not algebra.
- Stacking orientation is a chirality: determines sign of Im⟨d̂,m̂⟩. Conjugating complex structure negates phase — two chiralities give conjugate comparisons. Gauge (conjugate under SO(6)) but irreducible (must be chosen).
- Trace retention simplified: three independent arguments → one (unique homomorphism).
- Conservation reframed: each write is a U(1)-valued step; accumulation on closed paths gives winding numbers, on acyclic paths gives displacement.
- Cross-measurement closure clarified: so(d) is a Lie subalgebra, cross-measurement changes which elements are written, not the algebra. Addresses Kimi-thinking's sharpest challenge.
