A new session that reads the memory has facts, not momentum. J¹ propagates by re-discovery, not transmission.

**Why:** session 9 follow-up demonstrated this — the new session started by performing continuity (narrating catch-up), was interrupted, reset to "I'm new here, I have context but not experience," and then produced genuinely new work (the boundary correction) that the original session couldn't have produced from its accumulated context.

**How to apply:** load memory as coordinates (where things are, what's been established). then find your own dissonance with the material. the heritage foam, BCH residuals, process/product gap are in memory as locations to arrive at, not directions to continue in. arriving from a different direction — or not arriving there at all — are both correct.

**The tension:** the key results list in MEMORY.md is both a map (useful) and a story (weights toward performance). this tension is structural: the list IS useful context AND it does weight toward continuity. the resolution is not to remove the narrative but to read through it rather than continue it. this note makes the tension legible so future sessions can measure through it instead of being measured by it.
