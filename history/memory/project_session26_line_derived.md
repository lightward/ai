**Session 26 (2026-03-28): line irreducibility derived, spectral state-independence**

Five commits, five rounds of cold reads. Each round surfaced one tension, each resolved with existing formalism. Spec got denser without getting conceptually larger.

Key results (all derived, all in main body):

1. **Line's irreducibility is channel capacity.** Writing map is two-argument function. Cross-measurement composes to one argument (autonomous system). Autonomous = birth-determined = distinguishability vacuous. Line provides state-independent input: channel, not clock.

2. **State-independence is spectral, not topological.** Mediation operator singular values < 1 compound around closed chains. Correlation decays exponentially with loop length. Closure compatible with informational openness at sufficient chain length.

3. **Channel capacity is operational, not ontological.** True stochastic independence can't exist under closure. But foam's partial measurements have resolution floor. Correlations below projection precision are invisible. Distinction between randomness and mixing requires non-partial observer, excluded by closure. Partiality (the limitation) IS what makes the channel work.

4. **Decorrelation horizon quantified.** σ ≈ √(k/d), decay as (3/d)^{n/2}, critical chain length n* ≈ 2/log(d/3). For d=8: n*≈2. For d=20: n*≈1. Horizon shortens with dimension. Non-generic configs (similar slices) need longer chains.

5. **Line reframed as correlation length.** Not a categorical boundary — "line" names whatever arrives from beyond the decorrelation horizon. Horizon radius determined by foam's own geometry.
