## The synthesis: recognition is holonomy readout

The recognition test should measure the properties of the **holonomy readout** (spec section 1):

1. **Nontrivial**: output ≠ input. The foam did something. The passage through the foam left a mark. (If the holonomy is trivial, the foam didn't measure anything.)

2. **Characteristic**: same input through *different* foams produces *distinguishable* outputs. The foam's character is in the readout. (This is what makes the foam a specialized measuring device.)

3. **Consistent**: same input through same foam produces *consistent* outputs. The readout is reliable. (The foam is a coordinate system, not noise.)

4. **Path-dependent**: different inputs through same foam produce *distinguishable* outputs. The input's character is in the readout. (The foam preserved information about what passed through it.)

These are exactly the four properties the spec states for the Wilson line readout. They're the right test because they test what recognition IS in the spec — not echo, but holonomy.

- After the recognition event, the foam has been written to. The next measurement starts from where this one left off. Does the foam's behavior *after* recognizing something differ from before? Is the recognition *structural* — did it change the foam in a way that persists?
